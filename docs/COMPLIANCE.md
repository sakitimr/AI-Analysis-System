# 合规声明

> 字节跳动 CIS AI 全栈项目挑战 —— 竞品分析 Agent 协作系统

---

## 1. 信息采集合规

### 1.1 搜索工具

本系统使用 **DuckDuckGo Instant Answer API** 进行网络搜索：
- DuckDuckGo 提供公开 API 接口，无需绕过任何访问控制
- 搜索行为等同于用户在浏览器中使用搜索引擎
- 不采集任何需要登录或授权的内容

### 1.2 网页抓取

网页内容抓取遵守以下规范：
- 仅在用户发起分析任务时执行抓取
- 每次请求间隔 3-5 秒（`SCRAPE_DELAY = 3.0`），避免对目标站点造成压力
- 仅抓取公开可访问页面，不绕过任何访问控制或登录墙
- 仅提取页面正文摘要（最多 8000 字符），不下载完整页面或媒体资源
- 优先使用 DuckDuckGo 搜索结果摘要，仅在需要详细信息时才进行页面抓取

### 1.3 robots.txt 遵守

本系统使用的 `web_fetch` 工具基于 HTTP 请求，在可预见的范围内遵守目标站点的 robots.txt：
- 目标站点均为公开的科技媒体、产品官网、开发者社区
- 内容均为公开信息（产品功能、定价、用户评价）
- 不进行大规模爬取，单次分析 ≤ 30 个 HTTP 请求

## 2. 数据隐私与安全

### 2.1 数据存储

- 所有数据存储在本地 SQLite 数据库（`data/tasks.db`）和本地文件系统
- 不上传任何数据到云端服务
- 无用户个人信息收集

### 2.2 API 密钥管理

- API Key 通过 `.env` 文件管理，不纳入 Git 版本控制（`.gitignore` 已配置）
- 提供 `.env.example` 模板文件，不含真实密钥
- Doubao API Key 仅用于本课题项目，不外泄

### 2.3 报告安全

- 生成的竞品分析报告存储在本地 `data/reports/` 目录
- 报告内容不包含任何个人身份信息或敏感数据
- 报告中的数据分析结论均来自公开信息

## 4. 输出内容合规

### 4.1 竞品信息

- 所有竞品分析结论均来自公开可获取的信息
- 分析结果中标明数据来源 URL
- 不包含任何商业机密或内部信息
- 不歪曲或恶意贬低竞品

### 4.2 免责声明

每份生成报告末尾包含局限性声明：

> 本报告基于公开可获取的信息自动生成，数据采集于 {timestamp}。部分信息可能因数据源限制而不完整，仅供参考。
